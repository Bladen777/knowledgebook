SIZING
16px=12pt=1em=100%

Pixels (px) and points (pt) are static sizes and won't change with with web browser font size viewing. They also are unaffected by the parent font size settings. Pixels are also the most precise unit.

Percentage (%) and "em" are dynamic and will change with the web browser font size settings. However, the size is adding with the parent's font size and will result in a larger font size than intended.

Use "rem" (root em) as much as possible as it's size is unnafectd by the parent's font size setting.
